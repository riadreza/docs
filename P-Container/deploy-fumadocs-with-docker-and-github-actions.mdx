## Introduction
In this guide, we will walk through the steps to deploy <Icon icon="/icons/FumaDoc.svg" iconType="solid" />FumaDoc, an open-source documentation generator, in a Docker container and connect it to GitHub Actions for automated documentation generation and deployment.
<Info>
  ## Prerequisites
    Before we begin, ensure you have the following prerequisites in place:
    - A GitHub repository where you want to host your documentation.
    - Docker installed on your local machine or server.
    - Basic knowledge of Docker and GitHub Actions. 
</Info>
